실제 업무에서 검증된 프로젝트 경험

대규모 트래픽 광고 서버 부하 테스트 및 성능 개선

KT 나스미디어

실시간 광고 처리 최적화 및 안정성 검증을 통해 서비스 운영의 신뢰성을 높였습니다.

C/Linux HTTP/epoll k6 / hey

Problem & Goal

대규모 실시간 트래픽 환경에서 서버의 병목 구간을 파악하고, 운영 환경에 배포하기 전 최대 수용 가능한 성능 지표를 확보해야 했습니다.

Implementation

  • k6와 hey를 활용해 최대 490만 건 요청, 수만 RPS 규모의 스트레스 테스트 시나리오 설계 및 수행
  • Socket 통신 단계 병목을 분석하고 HTTP Keep-Alive 설정 및 epoll 기반 네트워크 I/O 최적화

Result

측정 API는 22,000 RPS 환경에서 평균 응답속도 66~210ms를 유지했고, 광고 요청은 최대 82,000 RPS 부하에서도 에러 없이 처리되었습니다.

OTT 플랫폼 실시간 광고 연동 시스템 개발

KT 나스미디어

쿠팡플레이 LIVE/VOD 광고 연동을 위한 VAST3 기반 송출 제어 및 예외 처리 로직을 구현했습니다.

C/Linux VAST3 MediaTailor

Problem & Goal

대형 OTT 스트리밍 환경에서 LIVE와 VOD 특성에 맞는 광고 송출 제어 및 Timeout 예외 처리 로직이 필요했습니다.

Implementation

  • C/Linux 환경에서 VAST3 프로토콜 분석 및 AWS MediaTailor 연동 규격 설계
  • LIVE 스트리밍과 VOD의 광고 노출 제어 로직 및 실시간 요청/응답 파이프라인 구축
  • 네트워크 지연에 대비한 Timeout 제어 및 과금 정확도를 위한 Reporting 로직 구현
